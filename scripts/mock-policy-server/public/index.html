Managed settings schema

Source: …

Status: …


			
Set the source with --schema <url|file-uri|path> or
				MANAGED_SETTINGS_SCHEMA. Default is
				./copilot-agent-runtime/schema/managed-settings-schema.json
				relative to the cwd. When loaded, the Managed Settings tab warns about
				top-level keys not declared in the schema.


			
Flow

			
Entitlements gate everything: the token and managed_settings
				are only fetched when chat_enabled is true, and the MCP
				registry only when the token enables mcp.
