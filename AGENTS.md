# VS Code Agents Instructions

This file provides instructions for AI coding agents working with the VS Code codebase.

For detailed project overview, architecture, coding guidelines, and validation steps, see the [Copilot Instructions](.github/copilot-instructions.md).

## Instruction Files

Specialized instructions for specific patterns and components are in `.github/instructions/`. These files contain detailed guidelines that you should consult when working with relevant code:
- Read instruction files when their description matches your task
- Follow the patterns and rules specified in applicable instruction files
- The `.github/prompts/` directory contains structured prompts for common engineering tasks

## Specialized Agents

The `.github/agents/` directory contains definitions for specialized sub-agents that can be invoked for specific tasks using the appropriate tooling.
