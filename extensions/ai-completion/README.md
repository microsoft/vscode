# AI Code Completion Extension

This extension provides intelligent code completion using open-source AI models.

## Features
- Context-aware code suggestions
- Multi-language support
- Low-latency performance
- Privacy-focused (no telemetry)

## Implementation
- Uses CodeLlama 7B for code generation
- Optimized context window handling
- Caching for improved performance
