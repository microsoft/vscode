# Markdown WYSIWYG Editor

**Notice:** This extension is bundled with Visual Studio Code. It can be disabled but not uninstalled.

## Features

This extension provides a custom WYSIWYG editor for Markdown files.

- Opens `.md` files in a custom editor
- Keeps Markdown source and visual editor content in sync
- Supports basic formatting actions (headings, lists, links, images, code)
- Supports Mermaid diagram blocks in the editing surface
