# Security Policy

**Please do not report security vulnerabilities through public GitHub issues.**

## VS Code core and fork patches

Report vulnerabilities in the VS Code core or fork-specific changes (BYOK pipeline, workbench chat, sessions UI) via [GitHub Security Advisories](https://github.com/sergey-zinchenko/vscode/security/advisories/new) on this repository.

## DIAL Chat Model Provider extension

Report vulnerabilities specific to the bundled DIAL extension (credential storage, OAuth/PKCE flows, log sanitization) using the policy in [extensions/dial-chat-model-provider/SECURITY.md](extensions/dial-chat-model-provider/SECURITY.md).

## Upstream Microsoft VS Code

For vulnerabilities in unmodified upstream code that also affect [microsoft/vscode](https://github.com/microsoft/vscode), you may additionally report to Microsoft via [https://aka.ms/SECURITY.md](https://aka.ms/SECURITY.md).
