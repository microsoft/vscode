Instructions for taking the perfect screenshot for your README

How to Take the Screenshot

Step 1: Open the Demo

Open demo-ai-app.html in your browser

Step 2: Set Up the Interface

  • Make sure the browser window is wide enough to show both components clearly
  • Load the demo content (click "Demo" button)
  • Enter a test API key in the API Key field
  • Ask a test question in the chat

Step 3: Take the Screenshot

  • Use Windows Snipping Tool or Snip & Sketch
  • Capture the entire interface showing both browser and chat
  • Make sure the URL bar shows "https://vnexpress.net"
  • Ensure the Vietnamese news content is visible
  • Include the chat history and input fields

Step 4: Save and Add to Repository

# Save the screenshot as: screenshots/vscode-ai-app-interface.png # Then commit and push: git add screenshots/vscode-ai-app-interface.png git commit -m "Add VSCode AI App interface screenshot" git push origin ai-app-modification

What Should Be Visible in the Screenshot

  • Left Side (80%): Browser component with URL bar and Vietnamese news content
  • Right Side (20%): AI chat interface with messages and input fields
  • URL Bar: Shows "https://vnexpress.net" with "Go" and "Demo" buttons
  • News Articles: 4 Vietnamese news articles about economy, infrastructure, etc.
  • Chat History: AI messages and status updates
  • API Key Field: Input field for OpenAI API key
  • Message Input: Text area for asking questions
