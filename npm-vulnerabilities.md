# NPM Vulnerabilities

## Arbitrary File Creation/Overwrite on Windows via insufficient relative path sanitization (High Development #8)

*   **Affected Component:** `tar` (npm)
*   **Location:** `package-lock.json`
*   **Severity:** High
*   **Description:** The `tar` package for npm is vulnerable to arbitrary file creation/overwrite on Windows due to insufficient relative path sanitization.

## Arbitrary File Creation/Overwrite due to insufficient absolute path sanitization (High Development #7)

*   **Affected Component:** `tar` (npm)
*   **Location:** `package-lock.json`
*   **Severity:** High
*   **Description:** The `tar` package for npm is vulnerable to arbitrary file creation/overwrite due to insufficient absolute path sanitization.

## PostCSS line return parsing error (Moderate Development #9)

*   **Affected Component:** `postcss` (npm)
*   **Location:** `package-lock.json`
*   **Severity:** Moderate
*   **Description:** The `postcss` package for npm has a line return parsing error.

## Koa Open Redirect via Referrer Header (User-Controlled) (Low Development)

*   **Affected Component:** `koa` (npm)
*   **Location:** `package-lock.json`
*   **Severity:** Low
*   **Description:** The `koa` package for npm is vulnerable to an open redirect via the Referrer Header.
